# ITWallpaper — Dev Desk

Живой дашборд разработчика в стиле VS Code / Cursor для **Wallpaper Engine**.

## Возможности
- **Часы-редактор** — текущее время в виде подсвеченного файла `clock.js`, с опциональным эффектом печати.
- **Погода** — данные в реальном времени (Open-Meteo, без ключа; опционально ключ OpenWeatherMap): температура, ощущается, макс/мин за день, влажность, точка росы, ветер + направление и порывы, давление, облачность, UV-индекс, видимость, осадки и их вероятность, рассвет/закат.
- **Уроки языков** — полностью data-driven виджет. Встроенные курсы: **тайский, испанский, английский, русский, японский, корейский**. Транскрипция зависит от языка интерфейса (кириллица для RU, латиница для остальных).
- **Заметки**, **Шпаргалка терминала**, **Горячие клавиши Windows**.
- **Быстрый доступ (apps)** — редактируемый борд ярлыков (ссылка на Steam-страницу автора по умолчанию; добавление/удаление/перемещение через ПКМ). Веб-ссылки открываются в браузере; локальные приложения показываются плитками (веб-обои WE не могут запускать внешние программы).
- **Дублирование модулей** — панели заметок и apps можно клонировать, каждая хранит своё содержимое.

## Настройка (в настройках обоев ⚙)
- **Основной язык** (интерфейс) и **Язык изучения** (уроки) выбираются независимо — EN / RU / ES / JA / KO / TH.
- **Темы**: VS Code Dark, Cursor, Midnight. **Акцентный цвет** из Wallpaper Engine.
- Выбор **цвета модулей** и **цвета текста** (с вводом по hex-коду).
- **Фон каждого виджета**: сплошной → полупрозрачный → отсутствует.
- **Перетаскивание и изменение размера** (режим редактирования), показ/скрытие виджетов, сворачиваемая перетаскиваемая кнопка меню.
- Всё сохраняется в `localStorage`; «Сбросить всё» возвращает значения по умолчанию.

## Архитектура
ES5, IIFE-модули, без сборки — работает по `file://` и внутри среды CEF Wallpaper Engine.
Новые языки добавляются папкой `data/languages/<id>/dataset.js` и записью в `data/languages/manifest.js`; логика ядра не меняется.
