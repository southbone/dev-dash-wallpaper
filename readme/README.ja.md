# ITWallpaper — Dev Desk

VS Code / Cursor 風のライブ開発者ダッシュボード（**Wallpaper Engine** 用）。

## 機能
- **エディター時計** — 現在時刻をシンタックスハイライトされた `clock.js` として表示（タイプライター演出はオプション）。
- **天気** — リアルタイムデータ（Open-Meteo、キー不要。OpenWeatherMap キーも任意）：気温、体感、最高/最低、湿度、露点、風＋風向と突風、気圧、雲量、UV指数、視程、降水量と降水確率、日の出/日の入。
- **言語レッスン** — 完全データ駆動のウィジェット。内蔵コース：**タイ語・スペイン語・英語・ロシア語・日本語・韓国語**。発音表記は表示言語に従う（ロシア語はキリル、他はラテン）。
- **メモ**、**ターミナル早見表**、**Windows ショートカット**。
- **クイックアクセス（apps）** — 編集可能なショートカット盤（既定はSteamページ。右クリックで追加/削除/並べ替え）。Web リンクはブラウザで開き、ローカルアプリはタイル表示（WE の Web 環境は外部プログラムを起動できません）。
- **モジュールの複製** — メモと apps パネルは複製でき、各々が独自の内容を保持します。

## カスタマイズ（設定 ⚙）
- **主言語**（UI）と **学習言語**（レッスン）を個別に選択 — EN / RU / ES / JA / KO / TH。
- **テーマ**：VS Code Dark、Cursor、Midnight。**アクセント色**は Wallpaper Engine から。
- **モジュール色**・**文字色**ピッカー（hex 入力対応）。
- **ウィジェット背景**：不透明 → 半透明 → なし。
- **ドラッグ＆リサイズ**（編集モード）、ウィジェットの表示/非表示、折りたたみ可能なドラッグ式メニューボタン。
- すべて `localStorage` に保存。「すべてリセット」で初期化。

## アーキテクチャ
ES5、IIFE モジュール、ビルド不要 — `file://` と Wallpaper Engine の CEF で動作。
