# ITWallpaper — Dev Desk

VS Code / Cursor 스타일의 라이브 개발자 대시보드(**Wallpaper Engine** 용).

## 기능
- **에디터 시계** — 현재 시간을 구문 강조된 `clock.js` 파일로 표시(타이핑 인트로 선택 가능).
- **날씨** — 실시간 데이터(Open-Meteo, 키 불필요. OpenWeatherMap 키 선택): 기온, 체감, 최고/최저, 습도, 이슬점, 바람＋풍향과 돌풍, 기압, 구름량, 자외선지수, 시정, 강수량과 강수확률, 일출/일몰.
- **언어 레슨** — 완전한 데이터 기반 위젯. 기본 코스: **태국어·스페인어·영어·러시아어·일본어·한국어**. 발음 표기는 인터페이스 언어를 따름(러시아어는 키릴, 그 외는 라틴).
- **메모**, **터미널 치트시트**, **Windows 단축키**.
- **빠른 실행(apps)** — 편집 가능한 바로가기 보드(기본값은 Steam 페이지; 우클릭으로 추가/삭제/이동). 웹 링크는 브라우저에서 열리고, 로컬 앱은 타일로 표시(WE 웹 런타임은 외부 프로그램을 실행할 수 없음).
- **모듈 복제** — 메모와 apps 패널을 복제할 수 있으며 각각 고유한 내용을 유지합니다.

## 사용자화 (설정 ⚙)
- **기본 언어**(인터페이스)와 **학습 언어**(레슨)를 독립적으로 선택 — EN / RU / ES / JA / KO / TH.
- **테마**: VS Code Dark, Cursor, Midnight. **강조 색**은 Wallpaper Engine에서.
- **모듈 색**·**글자 색** 선택기(hex 입력 지원).
- **위젯 배경**: 불투명 → 반투명 → 없음.
- **드래그 & 크기 조절**(편집 모드), 위젯 표시/숨김, 접을 수 있는 드래그형 메뉴 버튼.
- 모두 `localStorage`에 저장; "모두 초기화"로 기본값 복원.

## 아키텍처
ES5, IIFE 모듈, 빌드 불필요 — `file://` 및 Wallpaper Engine CEF에서 동작.
