# ITWallpaper — Dev Desk

แดชบอร์ดนักพัฒนาแบบสดสไตล์ VS Code / Cursor สำหรับ **Wallpaper Engine**

## คุณสมบัติ
- **นาฬิกาเอดิเตอร์** — เวลาปัจจุบันแสดงเป็นไฟล์ `clock.js` ที่ไฮไลต์ไวยากรณ์ พร้อมอินโทรแบบพิมพ์ดีด (เลือกได้)
- **สภาพอากาศ** — ข้อมูลเรียลไทม์ (Open-Meteo ไม่ต้องใช้คีย์; ใส่คีย์ OpenWeatherMap ได้): อุณหภูมิ, รู้สึกเหมือน, สูง/ต่ำของวัน, ความชื้น, จุดน้ำค้าง, ลม＋ทิศและลมกระโชก, ความกดอากาศ, เมฆ, ดัชนี UV, ทัศนวิสัย, หยาดน้ำฟ้าและโอกาสฝน, พระอาทิตย์ขึ้น/ตก
- **บทเรียนภาษา** — วิดเจ็ตขับเคลื่อนด้วยข้อมูลทั้งหมด คอร์สในตัว: **ไทย, สเปน, อังกฤษ, รัสเซีย, ญี่ปุ่น, เกาหลี** การถอดเสียงเป็นไปตามภาษาอินเทอร์เฟซ (ซีริลลิกสำหรับ RU, ละตินสำหรับอื่น ๆ)
- **บันทึก**, **สูตรคำสั่งเทอร์มินัล**, **คีย์ลัด Windows**
- **เข้าถึงด่วน (apps)** — บอร์ดทางลัดที่แก้ไขได้ (หน้า Steam ของผู้สร้างเป็นค่าเริ่มต้น; คลิกขวาเพื่อเพิ่ม/ลบ/ย้าย) ลิงก์เว็บเปิดในเบราว์เซอร์ ส่วนแอปในเครื่องแสดงเป็นไทล์ (เว็บรันไทม์ของ WE เรียกใช้โปรแกรมภายนอกไม่ได้)
- **ทำสำเนาโมดูล** — แผงบันทึกและ apps ทำสำเนาได้ แต่ละชุดเก็บเนื้อหาของตนเอง

## การปรับแต่ง (ตั้งค่า ⚙)
- เลือก **ภาษาหลัก** (อินเทอร์เฟซ) และ **ภาษาที่เรียน** (บทเรียน) แยกกัน — EN / RU / ES / JA / KO / TH
- **ธีม**: VS Code Dark, Cursor, Midnight. **สีเน้น** จาก Wallpaper Engine
- ตัวเลือก **สีโมดูล** และ **สีข้อความ** (ใส่รหัส hex ได้)
- **พื้นหลังต่อวิดเจ็ต**: ทึบ → โปร่งแสง → ไม่มี
- **ลากและปรับขนาด** (โหมดแก้ไข), แสดง/ซ่อนวิดเจ็ต, ปุ่มเมนูที่ย่อและลากได้
- ทุกอย่างบันทึกใน `localStorage`; "รีเซ็ตทั้งหมด" คืนค่าเริ่มต้น

## สถาปัตยกรรม
ES5, โมดูล IIFE, ไม่ต้องบิลด์ — ทำงานบน `file://` และในรันไทม์ CEF ของ Wallpaper Engine
